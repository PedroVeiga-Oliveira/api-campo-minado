# API Campo Minado

API REST desenvolvida em Node.js para uma plataforma de apostas baseada no jogo Campo Minado.

## Tecnologias Utilizadas
* Node.js (v24.15.0)
* Express.js
* PostgreSQL
* dotenv
* cors

## Instalação
```bash
git clone [https://github.com/seu-usuario/api-campo-minado.git](https://github.com/seu-usuario/api-campo-minado.git)
cd api-campo-minado
npm install
